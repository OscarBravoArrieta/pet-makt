    10 Creating an Angular App in the repo
        a. installing nx add @nx/angular
        b. If the instalatiion generates this error: The "@nx/angular" plugin doesn't support the existing TypeScript setup
           in the tsconfig.base.json file, set "declaration": false,
        c. To create the angular app, type the next command. appName is the application name
           nx g @nx/angular:app apps/appName

           the recomendated configuration:

        d. To run application
            nx serve pet-markt-web

    11. Adding tailwin.css to the Angular project
        a. modify:
           tsconfig.base.json-> removing "declaration": false, or setting "declaration": true,

           In the app angular, update the tsconfig.app.json:

            "types": ["node"],
            "composite": false,
            "emitDeclarationOnly": false,
            "declaration": true,
            "moduleResolution": "bundler",
            "module": "esnext",
            "lib": ["es2022", "dom"]

To resolve: The error "Missing parameter name at index 2: /**" in a Vite SSR.
The recommended fix for app.use('/**') is to change it to app.use('/{*path}'), in the server.ts file.
